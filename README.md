# my-popover

原生js+vue popover组件
在线演示：https://binbugs.github.io/my-popover/my-popover.html

- 实现点击显示popover框，点击外部关闭
- popover框跟随button居中定位
- 可设置文本框的内容和top bottom left right显示
- 文本框默认宽度，随内容居中撑开
